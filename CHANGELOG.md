## [1.0.0] - 2025-XX-XX

### Added

- Initial project setup with Next.js 14
- Firebase Authentication (Email/Password + Google OAuth)
- User profile system with onboarding
- Trip management system (create, browse, filter)
- Join request system for trips
- Real-time chat and messaging
- AI-powered Safety Checker using Gemini API
- Emergency SOS system with location sharing
- Admin dashboard for moderation
- User verification system
- Dark mode support
- Responsive design for all devices

### Tech Stack

- Next.js 14 (App Router)
- React 18 with TypeScript
- shadcn/ui + Tailwind CSS
- Firebase (Auth, Firestore, RTDB, Storage, Functions)
- Gemini 1.5 Pro API
- Framer Motion animations
- Zustand for state management
- SWR for server state

### Features

- 🔍 Find Travel Companions
- 💬 Real-time Chat
- 🛡️ AI Safety Checker
- 🚨 Emergency SOS
- 👥 User Verification
- 📊 Admin Dashboard
